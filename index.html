🚀 Setup & Hosting Guide
📱 Step 1: Create Telegram Bot
  1. Open Telegram and search @BotFather
  2. Send /newbot
  3. Enter your bot name: ZamSign
  4. Enter your bot username: zamsign_bot
  5. Copy the BOT_TOKEN provided
  6. Send /setmenubutton → select your bot → set button URL to your hosted app URL
  7. Send /setdomain → approve your GitHub Pages domain
💻 Step 2: Host on GitHub Pages
  1. Create a GitHub account at github.com
  2. Create a new repository: zamsign-miniapp
  3. Upload this index.html file to the repo root
  4. Go to Settings → Pages
  5. Under Source, select main branch → Save
  6. Your app URL will be: https://yourusername.github.io/zamsign-miniapp
🔗 Step 3: Link Bot to Mini App
  1. In BotFather, send /newapp
  2. Select your bot
  3. Enter app title: ZamSign
  4. Paste your GitHub Pages URL
  5. Users can now open ZamSign from your bot
⚙️ Environment Variables (Render.com)
BOT_TOKEN=your_bot_token
PORT=10000
RENDER=1
RENDER_EXTERNAL_URL=https://yourapp.onrender.com
